Northcoders News API

  • /api/topics
  • /api/articles
  • /api/users
  • /api/articles/:article_id
  • POST comment
  • PATCH article
  • DELETE comment

GET /api/topics

Returns an array of all topics.

{
  "topics": [
    { "slug": "coding", "description": "Code is love" },
    { "slug": "football", "description": "Footie fanatics" }
  ]
}

GET /api/articles

Returns an array of articles with comment count, sorted by date.

{
  "articles": [
    {
      "article_id": 1,
      "title": "Cats are great",
      "topic": "cats",
      "author": "jessjelly",
      "created_at": "2020-01-01T00:00:00.000Z",
      "votes": 5,
      "comment_count": 2
    }
  ]
}

GET /api/users

Returns a list of all users.

{
  "users": [
    { "username": "butter_bridge", "name": "Jonny", "avatar_url": "" }
  ]
}

GET /api/articles/:article_id

Returns a specific article by ID.

POST /api/articles/:article_id/comments

Adds a new comment to an article.

Example request body:

{
  "username": "butter_bridge",
  "body": "Great article!"
}

PATCH /api/articles/:article_id

Updates the votes of an article.

Example request body:

{ "inc_votes": 1 }

DELETE /api/comments/:comment_id

Deletes a comment by ID. Responds with 204.
